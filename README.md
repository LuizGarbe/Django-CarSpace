# Django Car Gallery 🚗

Um projeto de galeria de fotografias automotivas desenvolvido em Django. A aplicação permite o gerenciamento de imagens, categorias e usuários, focando em modelos de alta performance como Porsche, Audi RS6 e Mercedes AMG.

## 🚀 Tecnologias Utilizadas
- **Python 3.11**
- **Django 4.1**
- **SQLite** (Banco de dados local)
- **Pillow** (Processamento de imagens)
- **Python-dotenv** (Gerenciamento de variáveis de ambiente)
