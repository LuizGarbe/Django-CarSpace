# 🏎️ Django CarSpace

O **CarSpace** é uma aplicação web para entusiastas do automobilismo, desenvolvida em Django. O projeto funciona como uma galeria digital onde fotos de coleções exclusivas de carros (como Porsche, Audi RS6 e Mercedes AMG) podem ser gerenciadas e visualizadas com metadados detalhados.

Este projeto foi evoluído a partir de uma base educacional da Alura, com personalizações de tema, banco de dados e lógica de usuários.

## 🛠️ Tecnologias e Ferramentas
- **Python 3.11**
- **Django 4.1**
- **SQLite** (Banco de dados local)
- **Pillow** (Processamento de imagens)
- **Python-dotenv** (Segurança de credenciais)

## ✨ Funcionalidades
- [x] Galeria dinâmica de fotografias automotivas.
- [x] Filtros por categorias de coleções.
- [x] Painel Administrativo Django para gestão de conteúdo.
- [x] Sistema de autenticação e vínculo de fotos por usuário.
- [x] Variáveis de ambiente protegidas.
